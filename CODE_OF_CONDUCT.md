# Code of Conduct

## Our standards

Contributors are expected to:

1. Be respectful and professional in all project spaces.
2. Give and accept constructive feedback.
3. Focus discussions on technical issues, not people.
4. Avoid harassment, discrimination, or abusive behavior.

## Scope

This code of conduct applies to this repository, issues, pull requests, and project discussions.

## Enforcement

Report conduct concerns privately through GitHub maintainers. Maintainers may remove comments, close threads, or restrict access for behavior that violates this policy.
